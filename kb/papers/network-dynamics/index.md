# Network Dynamics

## Scope

Network dynamics concerns the collective behaviour of neurons coupled through excitatory and inhibitory synapses. Individual neurons fire spikes, but coupled populations generate oscillations, transient synchrony, attractor states, and chaotic trajectories: phenomena with no single-neuron counterpart. These emergent properties are the necessary bridge between cellular mechanisms and systems-level computation.

## Relevance

Functional brain states, memory retrieval, sensory binding, sleep-wake transitions, pathological seizures, are expressed as network-level dynamical phenomena, not single-neuron properties. Neuromorphic systems that interface with or replicate these functions must produce the same dynamical signatures. Attractor network theory provides one of the few tractable mathematical frameworks directly connecting synaptic connectivity structure to computational function.
